Checkpoint 01 - HTML/CSS

O que é HTML

O acrônimo HTML significa em inglês: Hyper Text Markup Language. Para gente aqui fica: Linguagem de Marcação de Hipertexto. Por trás das palavras Hipertexto e Marcação tem muita história e guardam a real essência da função do HTML, onde a Semântica, que nada mais é do que a organização da informação usando HTML.

O que é CSS

CSS significa Cascading Style Sheets. O Cascading Style Sheets (CSS) é uma linguagem utilizada para definir a aparência de documentos que adotam para o seu desenvolvimento linguagens de marcação. O CSS define como serão exibidos os elementos contidos no código de um documento e sua maior vantagem é efetuar a separação entre o formato e o conteúdo de um documento.

A Web Semântica

A Web Semântica é nada mais nada menos que uma web com toda sua informação organizada de forma que não somente seres humanos possam entende-la, mas principalmente máquinas. As máquinas nos ajudarão em tarefas que hoje fazemos manualmente.
